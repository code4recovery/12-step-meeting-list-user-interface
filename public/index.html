TSML UI

TSML UI (12 Step Meeting List User Interface) is an open source interactive recovery meeting finder that can be embedded on any website.

12 Step Meeting List

Using the 12 Step Meeting List WordPress plugin, use this built-in shortcode to embed it in a WordPress post or page. 

[tsml_ui]
Demo: Santa Cruz

Custom JSON feed

If you can generate a Meeting Guide-spec JSON feed, say, from a custom database, you can plug that info in to the format below. To enable maps, add a Mapbox API access token (or remove that parameter). Be sure to add a proper time zone for your location. 

<div
  id="tsml-ui"
  data-src="/path/to/your.json"
  data-mapbox="pk.your.mapbox.access.token.goes.here"
  data-timezone="America/Los_Angeles"
></div>
<script src="https://tsml-ui.code4recovery.org/app.js" async></script>
Demo: San Francisco & Marin

More information is available on the project page on GitHub, including advanced configuration options. To get help, please file an issue.
